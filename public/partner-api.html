Nock API Reference

The Nock API gives partners programmatic access to the Nock agents on Robinhood Chain — swaps, yield, tokenized stocks, stock-backed lending, and perps. Read live market data, or build a transaction and sign it yourself. Nock never holds your keys.

Base URL  https://app.nockfi.io/api/v1
Chain  Robinhood Chain (4663)
Auth  API key
Custody model. Every endpoint is read-only on our side. Endpoints that create an action return an unsigned transaction (or an ordered list of them); your application signs and broadcasts. Keys, approvals, and enforcement stay entirely with you.

Authentication

Every request must carry your API key. Requests without a valid key return 401 — no endpoint is public.

Pass your key in the x-api-key header (or Authorization: Bearer <key>). We issue one key per partner and store only its hash. Requests are rate-limited per key; contact us to raise limits.

RequestcURL
curl https://app.nockfi.io/api/v1/yield/markets \
  -H "x-api-key: npk_live_your_key_here"

Swap Agent

GET/swap/quoteExecutable

Returns the best-price route between two tokens on Robinhood Chain (routed via 0x), including an unsigned transaction to execute the swap. Every quote carries a verified flag — false means at least one token is an unvetted/impersonator contract.

Query parameters
ParameterTypeRequiredDescription
fromTokenstringrequiredToken to sell — a symbol (USDG) or a contract address.
toTokenstringrequiredToken to buy — symbol or contract address.
amountstringrequiredHuman amount of fromToken, e.g. "100".
takerstringoptionalWallet that will sign the transaction. Include it to get an executable transaction.
Request
GETcURL
curl "…/v1/swap/quote?fromToken=USDG\
&toToken=NOCK&amount=100\
&taker=0x6690…Aab1" \
  -H "x-api-key: npk_live_…"
Response 200
JSON
{
  "fromSymbol": "USDG",
  "toSymbol": "NOCK",
  "toAmount": "5404.03",
  "exchangeRate": "1 USDG = 540.4 NOCK",
  "liquidityAvailable": true,
  "verified": true,
  "sellTokenAddress": "0x5fc5…d168",
  "sellTokenDecimals": 6,
  "transaction": {
    "to": "0x0000…2734",
    "data": "0x…",
    "value": "0", "gas": "250000"
  }
}

Lending — borrow against stocks

GET/collateral/markets

Which tokenized stocks can back a USDG loan, with live borrow APY, max LTV (LLTV), and available liquidity.

Query parameters

None.

GET/collateral/positions

A wallet's open loans: debt owed, collateral value, and liquidation price.

Query parameters
ParameterTypeRequiredDescription
addressstringrequiredWallet address.

GET/collateral/borrowExecutable

Post tokenized-stock collateral and borrow USDG (e.g. post TSLA, borrow USDG). Returns the ordered step transactions plus any required ERC-20 approval, and the resulting LTV, borrow APY, and liquidation price so you can gate on risk. Over-leveraged requests are rejected with the safe maximum.

Query parameters
ParameterTypeRequiredDescription
addressstringrequiredBorrowing wallet.
stockstringrequiredCollateral stock symbol, e.g. TSLA.
borrowUsdstringrequiredUSDG amount to borrow.
collateralAmountstringoptionalStock units to post. Defaults to the wallet's full balance.
Response
JSON200
{
  "kind": "stock-borrow", "stockSymbol": "TSLA",
  "approval": { "tokenAddress": "0x…", "amountRaw": "…", "spender": "0x…" },
  "steps": [ { "to": "0x…", "data": "0x…", "value": "0" } ],   // sign in order
  "usdgAmount": "…", "borrowApyPct": 6.1,
  "ltvUtilizationAfterPct": 42.0, "liquidationPriceUsdAfter": …
}

Perps Agent

GET/perps/markets

Live perpetual-futures market data on Robinhood Chain (via Lighter): mark price, hourly funding, open interest, 24h volume, and max leverage. Read-only — live order execution is jurisdiction-gated and not exposed via the API.

Query parameters
ParameterTypeRequiredDescription
symbolstringoptionalFilter to one market, e.g. ETH. Omit for the top markets by volume.
Response
JSON200
{ "markets": [
  { "asset": "ETH", "markPrice": 1831.61, "fundingRatePctHourly": 0.0096,
    "openInterest": 49610, "dailyVolumeUsd": 286985432, "maxLeverage": 20 }
] }
